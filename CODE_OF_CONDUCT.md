# Code of Conduct

This project follows the
[Contributor Covenant Code of Conduct, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html).

## TL;DR

- Be respectful in issues, pull requests, code reviews, and the
  ARISE Discord channels.
- Assume good faith; ask clarifying questions before correcting.
- No harassment, no discrimination, no personal attacks.
- Industrial robotics is a small community — be the colleague you'd
  want to work with on a production cell at 2 a.m.

## Scope

This Code of Conduct applies to all project spaces: GitHub issues
and pull requests, the public ARISE Discord channels associated with
this experiment, and any in-person events (workshops, demonstrator
visits) that involve maintainers in their role on this project.

## Reporting

Suspected violations can be reported privately to **tech@ampero.it**
with the subject line `CONDUCT — hermes-odoo-adapter`. Reports are
treated confidentially; maintainers will respond within 7 days.

For severe or escalating situations involving the broader ARISE
ecosystem, please also contact the ARISE Tech Coordinator
(Francisco Melendez) per the support process documented in the D4
materials.

## Enforcement

The maintainers will follow the
[Contributor Covenant Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)
to determine consequences for any action they deem in violation of
this Code of Conduct. The typical escalation is: private
correction → public warning → temporary ban → permanent ban.
